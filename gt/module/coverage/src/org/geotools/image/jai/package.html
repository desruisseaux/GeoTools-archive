A set of Java Advanced Imaging operations. All operations defined here work on J2SE's {@link java.awt.image.RenderedImage rendered images}; there is no OpenGIS's {@linkplain org.opengis.coverage.grid.GridCoverage grid coverage} depencencies here.

Before to use any Geotools's JAI operations, it is necessary to invoke {@link org.geotools.image.jai.Registry#init} at least once.
